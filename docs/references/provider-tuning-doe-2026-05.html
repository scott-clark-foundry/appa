Methodology

Script: scripts/doe_provider_tuning.py · Summary tables: scripts/doe_summarize.py · This report: scripts/doe_html_report.py

Capture layers per call (all joined by trace_id in Logfire):

  1. pydantic-ai agent spans via logfire.instrument_pydantic_ai() — gen_ai.* attributes
  2. Python-object outgoing request via Hooks(before_model_request=...) — vault/doe/<slug>/<cell>/<prompt>.requests.jsonl
  3. Raw HTTPS payloads via logfire.instrument_httpx(capture_all=True) — request + response bodies in Logfire

What we did NOT measure: quality (human-graded in a follow-up pass), server-side tool fees (OpenAI $30/1k web searches etc. — shows on the bill, not in our token cost), variance (each cell ran once).

Re-run: uv run python scripts/doe_provider_tuning.py --dry-run for the plan, drop --dry-run to execute. rm -rf vault/doe/ first for a clean re-run.

Raw CSV: vault/doe/runs.csv · Per-call metadata: vault/doe/<slug>/<cell>/<prompt>.meta.json · Outgoing-request trace: <prompt>.requests.jsonl · Logfire spans joinable by trace_id.
